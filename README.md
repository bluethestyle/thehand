# THE HAND — 페이지형 디지털 메뉴판

사케 전문 이자카야 **더핸드(THE HAND)** 의 태블릿 웹 메뉴판.
종이 바인더 → 좌우로 넘기는 **페이지형(booklet) 디지털 메뉴판** 으로 전환하는 프로젝트.

피그마 핸드오프(`scratch/svgview-v3-pages.html`)의 v3 "페이지형" 시안을 실제 동작하는 앱으로 구현합니다.

## 화면 (v3 페이지형 시안)

**손님용 (좌우 스와이프 + dot 인디케이터, 768×1024 세로)**
1. 니혼슈 — 등급 카테고리 + 8종/페이지 자동분할
2. 수입사 한정주 이미지 페이지 — 자유 배치 스티커
3. 이벤트 이미지 페이지
4. 취향 지도 — 달콤↔깔끔 · 화려↔은은

**관리용 (메뉴 내 버튼 + 비밀번호로 진입)**
1. 페이지 보드 — 순서/삽입/숨김
2. 페이지당 항목 수 ↔ 글자 크기(자동 비례)
3. 이미지 페이지 스티커 편집기
4. 메뉴 항목 관리 — 판매중 / 일시품절 / 닫힘 + 보관함
5. 비밀번호 관리 — 최초 설정 / 변경

## 디자인 토큰

Pruf 디자인 시스템 기반 (warm neutral + amber). 4색 스타일 시스템:
화사한(薫酒 `#d4618c`) · 산뜻한(爽酒 `#5dade2`) · 든든한(醇酒 `#e2a04f`) · 깊은(熟酒 `#8b6f47`).

자세한 사양은 `design-reference/` 의 원본 핸드오프 시안을 참조.

## 기술 스택

- **프론트/배포**: Next.js 16 (App Router, TS) → Vercel
- **DB/실시간/인증**: Supabase (Postgres + Realtime), `thehand_` 접두사 + RLS(손님 읽기전용)
- **관리자**: 메뉴 화면 ⚙ → 비밀번호(최초 설정/변경, scrypt 해시) → 서버 라우트(service_role) 쓰기

## 실행

Supabase 미연결 상태에서도 내장 시드로 미리보기가 됩니다.

```bash
npm install
npm run dev        # http://localhost:3000
```

실제 DB 연결·배포는 [SETUP.md](SETUP.md) 참고.

✅ 손님 5개 화면 + 관리자 5개 화면 구현·검증 완료. Supabase 연결 시 실시간 운영.
